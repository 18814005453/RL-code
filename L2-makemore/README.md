# L2 makemore (Bigram)

纯统计概率模型：数数 → 概率 → 采样 → 评估。没有神经网络，没有梯度下降。

## 运行

```bash
python3 bigram.py
```

改 `USE_CHINESE = True` 切换到中文名数据集。

## 结果

| 数据集 | 字表 | NLL | Perplexity |
|--------|------|-----|------------|
| 英文名 (32033) | 28 | 2.45 | 11.6 |
| 中文名 (9523) | 461 | 4.47 | 87.5 |

## 做了什么

1. 统计相邻字符对次数 → `N[28,28]`
2. 归一化变概率 → 每一行 = 下一个字符的概率分布
3. 按概率随机采样 → 生成新名字
4. 负对数似然评估 → NLL/Perplexity
